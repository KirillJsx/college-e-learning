# college-e-learning


Учимся превращать данные в HTML-разметку при помощи циклов и шаблонов.

Перед выполнением задания создайте новый проект (папку) с тремя файлами: `index.html`, `styles.css`, `app.js`.

Самостоятельно реализуйте базовые вещи:
- HTML-структуру с заголовком `<h1>` (например, «Сетевые подключения»).
- Базовые CSS-стили: центрирование контента, аккуратное оформление таблицы.

---
#### Уровень 1

**Задача:** отрисовать таблицу сетевых подключений из готового массива **без использования строковых шаблонов** - только с помощью методов создания DOM-узлов. Мы уже это реализовывали когда проходили `localStorage`

Подготовленный `html` найдете на гитхабе что бы Вам было легче.

**Данные:**
```js
const servers = [
    { proto: "TCP", local: "192.168.1.5:443", remote: "10.0.0.1:80", state: "ESTABLISHED" },
    { proto: "UDP", local: "192.168.1.5:53", remote: "8.8.8.8:53", state: "LISTENING" },
    { proto: "TCP", local: "192.168.1.5:22", remote: "192.168.1.10:54321", state: "TIME_WAIT" }
];
```
**Требования:**
1. В html уже должна быть пустая таблица с `<tbody id="table-body"></tbody>`.
2. Напишите функцию `renderTable(data)`.
3. Внутри функции очистите `tbody` (например, через `innerHTML = ""` - это допустимо только для очистки).
4. Пройдитесь по массиву `data` с помощью `forEach`.
5. Для каждого объекта:
    - Создайте строку таблицы `document.createElement("tr")`.
    - Создайте четыре ячейки `document.createElement("td")`.
    - Заполните `textContent` каждой ячейки соответствующими данными (`proto`, `local`, `remote`, `state`).
    - Добавьте ячейки в строку через `appendChild`.
    - Добавьте строку в `tbody` через `appendChild`.
6. Вызовите `renderTable(servers)` после загрузки страницы.

**Задача:** добавить цвет фона для строки в зависимости от состояния соединения.

#### Уровень 3

**Задача:** добавить поле ввода, которое фильтрует таблицу по IP-адресам (поля `local` или `remote`).

**Требования:**
1. Добавьте над таблицей HTML-элемент: `<input type="text" id="search" placeholder="Поиск по IP...">`.
2. Напишите обработчик события `input` для этого поля.
3. В обработчике:
    - Получите значение из поля (`e.target.value`), приведите к нижнему регистру.
    - Используйте метод массива `.filter()`, чтобы создать новый массив, содержащий только те соединения, у которых `local` **или** `remote` **включают** введённый текст (проверка через `.includes()`).
    - Вызовите `renderTable(отфильтрованный_массив)`.
4. Если поле поиска пустое - должна отображаться полная таблица (исходный массив).

> **Подсказка**: Не забываем про console.log() что бы смотреть что выводится.
> Про метод `filter()` для работы с массивами - он создает новый массив при нахождение тех или иных данных в нем:

```js
arr = ["a","b","c"]
input = "a"
newArr = arr.filter(item => { return item.includes(input) })
```

> **Еще подсказка**: так как у нас массив с объектам, для поиска через `includes()` у нас обращение должно идти не только к массиву, но и к ключу.

#### Уровень 4 

**Задача:** добавить кнопку «X» (закрыть) в каждую строку и реализовать удаление соединения из исходного массива.

**Требования:**
1. При создании строки (в `forEach`) добавьте пятую ячейку (`<td>`), в которую поместите кнопку:
```js
const delBtn = document.createElement("button");
delBtn.textContent = "X";
delBtn.className = "btn-close";
// учтите что так мы создаем кнопку, но кнопка должна быть внутри td, значит нужно перед кнопкой создать td 
```
2. Чтобы знать, какую строку удалять, добавьте каждой строке `<td>` атрибут `data-index`. Для этого в `forEach` используется второй аргумент - `index`.  
    Пример: `row.dataset.index = index;` (или `row.setAttribute("data-index", index)`).
3. **Не вешайте обработчик на каждую кнопку отдельно.** Вместо этого повесьте **один** обработчик `click` на элемент `<tbody>`.
4. В обработчике:
    - Проверьте, что клик был по кнопке: `if (e.target.classList.contains("btn-close"))`.
    - Найдите родительскую строку `tr` с помощью `e.target.closest("tr")`.
    - Извлеките индекс из `dataset.index` этой строки.
    - Удалите элемент из **исходного массива** `servers` с помощью `.splice(index, 1)`.
    - Перерисуйте таблицу, вызвав `renderTable(servers)` (или, если хотите усложнить, `renderTable(отфильтрованный_массив)`, но в рамках уровня достаточно полной перерисовки).

> **Важно:** нигде не используйте `innerHTML` (кроме очистки контейнера в начале функции `render`) и не формируйте html-строки. Только `createElement`, `textContent`, `classList`, `appendChild` и работа с `dataset`.
> **Подсказка**: можно сделать отдельную функцию слушателя и вешать ее в цикле на каждую кнопку. Но тогда логика выше не подойдет и потребуется другой подход к реализации удаления. (кстати такой подход через события кнопки в цикле может быть проще, но вариант через e.taget будет просто концептуально новый для вас.)
